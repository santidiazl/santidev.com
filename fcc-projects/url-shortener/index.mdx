A microservice that allows users to save URLs through a POST request to /api/shorturl and then retrieve and be redirected to original URL by a GET request to /api/shorturl/<short_url>
The live demo in link below will pass all 4 of freeCodeCamp's <a href="https://www.freecodecamp.org/learn/back-end-development-and-apis/back-end-development-and-apis-projects/url-shortener-microservice" target="_blank">required user stories</a>.
