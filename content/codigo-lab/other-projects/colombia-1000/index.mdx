A Front End React project that integrates with the <a href='https://dev.socrata.com/' target='_blank'>Socrata</a> and <a href='https://www.mediawiki.org/wiki/API:Main_page' target='_blank'>Wikipedia</a> APIs
and displays Colombia's 1000 largest companies by revenue. Allows for searching by name and filtering by region.
